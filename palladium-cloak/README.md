# Palladium Cloak - Static HTML Landing Page

This is a static HTML version of the Black Affiliate Marketing landing page, designed for use with Palladium cloaking system.

## Structure

- `index.html` - Main landing page with all sections
- All images should be placed in `/img/black-affiliate-marketing/` directory relative to the HTML file

## Features

- Pure HTML + Tailwind CSS (via CDN)
- Responsive design
- Interactive FAQ section
- Image modal gallery for testimonials
- All sections from the original React version:
  - Hero Section
  - Course Target Section
  - Providing Section
  - Phase System Section
  - Testimonials Section
  - Pricing Section
  - FAQ Section
  - Two Options Section
  - Footer

## Setup

1. Place all images in the correct directory structure:
   ```
   /img/black-affiliate-marketing/
     - hero.webp
     - arrow.webp
     - 1.webp, 2.webp, 3.webp, 4.webp
     - ico-1.avif through ico-12.avif
     - community-icon-1.webp through community-icon-3.webp
     - testimonial-1.webp through testimonial-9.webp
     - video-4.mp4, video-5.MP4
     - linkedin.avif, instagram.avif, facebook.avif
   ```

2. Update form links: Replace `#form` with your actual form URL

3. Deploy to your hosting or use with Palladium cloaking system

## Customization

- Update form action URLs in all "Get It Now" and "SECURE YOUR SPOT NOW" buttons
- Modify pricing, text, or images as needed
- Adjust Tailwind classes for styling changes

## Browser Support

Works in all modern browsers that support:
- CSS Grid
- Flexbox
- ES6 JavaScript
